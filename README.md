# Mbogo Foundation Management System

A comprehensive web application for community foundation management, featuring member registration, meeting coordination, and administrative tools.

## Features

- Member registration with GPS location capture
- Meeting management and attendance tracking
- SMS messaging system
- Administrative dashboard
- Database management tools

## Tech Stack

**Backend:**
- Flask (Python)
- SQLAlchemy
- SQLite

**Frontend:**
- React
- Modern CSS with gradients

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
python main_app.py
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Usage

1. Access the application at `http://localhost:3000`
2. Register new members through the registration form
3. Manage meetings and track attendance
4. Use admin panel for system administration
